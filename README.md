# Vim  prepackaged colorscheme.
### Vim comes with 16 prepackaged colorscheme which is well in **_Vi IMproved 7.4_** since I'm using this version. You can hook with this colorscheme simply use 
```:colorscheme <colorname> ```

### List of colorscheme
- darkblue
- default
- delek
- desert 
- elflord
- evening
- industry 
- koehler
- morning
- murhpy
- pablo
- peachpuff
- ron
- shine
- slate
- torte 
- zellner
